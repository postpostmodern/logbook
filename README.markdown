### Logbook is a personal, online journal in plain text.

### You know, like "Captain's Log"?

### Installation:

- Set a password in `app/config.yaml`.
- Set your webserver's document root to `public/`.
- Write things that are on your mind.

### Site structure:

	.
	|-- app
	|   |-- config.yaml
	|   `-- load.php
	|-- content
	|   `-- 2008
	|       `-- june
	|           `-- 25.txt
	`-- public
	    |-- css
	    |-- img
	    |-- js
	    `-- index.php
